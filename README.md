**AgentVendi v1** is a professional-grade, enterprise-ready AI agent creation platform. It transforms the complex task of agent engineering into a seamless "Vending Machine" experience—offering biological cognitive matching, local-first AI suggestions, and universal IDE rule generation.

---

## 🏗️ Core Features

- **🎰 7-Tab Creation Wizard**: Deeply configure Identity, Behavior, Knowledge, Guardrails, Skills, and Cognitive profiles.
- **🎓 Agent Masterclass**: Built-in 12-step mastery guide for engineering high-performance coding agents.
- **🐍 Vendi Script (Import)**: Python-like syntax for structured agent generation and CI/CD automation.
- **🧠 Local AI suggestions**: Uses **Transformers.js** (in-browser) to provide semantic skill matching without backend calls.
- **🏢 Enterprise Intranet Mode**: Centrally manage Ollama endpoints and global mandatory guardrail policies.
- **📦 Multi-IDE Export**: Single-click bundle generation for Cursor, Windsurf, Claude Code, Aider, Cline, and Copilot.

## 🚦 Getting Started

### Prerequisites
- Node.js (v18+)
- Ollama (optional, for local chat AI)

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/Santhoshnadella/agentvendiv1.git
   cd agentvendiv1
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```

## 🏛️ Architecture & Stack

- **Frontend**: Vite + Vanilla JS (Modern Neon-Glass UI).
- **Backend**: Node.js + Express + Better-SQLite3.
- **Local AI**: Transformers.js (Feature: Semantic matching).
- **Server AI**: Ollama / Enterprise LLM endpoints.
- **Testing**: Vitest for core engine & API reliability.
