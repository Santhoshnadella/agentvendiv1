# AgentVendi v1 — The Ultimate Product Guide 🎰

Welcome to **AgentVendi**, the professional-grade AI agent creation platform. Whether you're building for a public marketplace or a secured enterprise intranet, AgentVendi provides the "Vending Machine" experience for AI agents.

## 🚀 The 12-Step Agent Mastery Framework

AgentVendi is built upon the following industry-standard framework for high-performance coding agents:

1.  **Define the Agent’s Scope**: Start by narrowing what your agent should do (Debug, Generate, Refactor, or Consult).
2.  **Choose the Core Model**: Optimize for context length and tool-calling (GPT-4o, Claude 3.5, or Local llama3).
3.  **Design the Agent Architecture**: Use the internal Reasoning Loop (ReAct pattern) for complex tasks.
4.  **Add Tools (Critical)**: Equip your agent with file access, test runners, and code executors.
5.  **Implement the Agent Loop**: A robust while-loop that reasons, acts, and parses output.
6.  **Context Management**: Use RAG, AST parsing, and embeddings to keep the agent focused.
7.  **Execution + Feedback Loop**: The "Magic" of write-run-error-fix-repeat.
8.  **Safety & Guardrails**: Sandbox execution and strict command filtering (preventing `rm -rf`).
9.  **Evaluation System**: Measure performance with benchmarks like HumanEval.
10. **Deployment**: Export to Cursor, Windsurf, Claude Code, Aider, or Cline.
11. **Tech Stack Optimization**: Leverage FastAPI, Vector DBs, and Docker.
12. **Iteration**: Continuous improvement through feedback and versioning.

## 🛠️ Platform Features

### 1. The 7-Tab Wizard
A guided experience to build complex agents without touching a line of code:
- **Selection**: Single or Multi-agent orchestration.
- **Behavior**: Tone, creativity, and autonomy calibration.
- **Knowledge**: RAG-ready documentation and domain expertise.
- **Role**: Persona and objective definition.
- **Guardrails**: Global and custom safety rules.
- **Skills**: Local-AI powered semantic skill suggestions.
- **Cognitive**: Interactive "Therapy-mode" to match your biological thinking style.

### 🏢 Enterprise Intranet Mode
AgentVendi is designed to be the "Internal AI Vending Machine" for companies:
- **Local LLM Support**: Connect to internal Ollama/LocalAI endpoints.
- **Mandatory Guardrails**: Admins can enforce company policies across all agents.
- **Air-Gapped Ready**: 100% local operation with no external data leakage.

### ⚡ Script-to-Agent (Power User Feature)
Paste any agent description or markdown "script" into the **Masterclass Portal**, and AgentVendi will use **Transformers.js** to parse the text and auto-configure the machine.
